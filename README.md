# 📊 Finance & Analytics Portfolio
### Python | SQL | Financial Analysis | Data Visualization

A collection of financial analytics projects built with Python and SQL,
demonstrating skills in data acquisition, analysis, and reporting.

---

## 🏗️ Projects

### 1. Stock Portfolio Analyzer
**Tools:** Python, yfinance, pandas, matplotlib

Analyzes 5 stocks over a 16-month period using real market data.

**Key Features:**
- Pulls live stock data via Yahoo Finance API
- Calculates daily & cumulative returns
- Computes Sharpe Ratio for risk-adjusted performance
- Generates correlation heatmap for diversification analysis
- Produces a full portfolio summary report

**Key Finding:** JPM delivered the highest risk-adjusted return (Sharpe: 1.045)
outperforming GS despite identical raw returns, due to lower volatility.

---

### 2. Automated Financial Report Generator
**Tools:** Python, yfinance, openpyxl, reportlab

Automatically pulls company financial statements and generates
a formatted Excel workbook and PDF report.

**Key Features:**
- Fetches 4 years of income statement & balance sheet data
- Calculates key metrics: gross margin, EBITDA margin, net margin
- Exports a multi-sheet formatted Excel report
- Generates a professional PDF with analyst commentary

**Key Finding:** Dell's net income nearly tripled from $2.4B to $5.9B
over 4 fiscal years, driven by AI infrastructure demand.

---

### 3. SQL Business Dashboard
**Tools:** Python, SQLite, SQL, matplotlib

Built a relational database from scratch and wrote SQL queries
to answer business questions, visualized in an executive dashboard.

**Key Features:**
- Designed 3-table relational database (customers, products, sales)
- Wrote JOIN queries to combine data across tables
- Used GROUP BY aggregations to calculate revenue by customer,
  category and region
- Built an executive dashboard with KPI cards and 4 charts

**Key Finding:** Software was the top revenue category, East region
led all geographies, with notable revenue spikes in Q2 and Q3
suggesting budget-cycle driven purchasing behavior.
